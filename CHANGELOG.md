## [2.1.1] - 2026-03-04

### Added
- تبويبات الأقساط الثلاثة في إدارة التحصيل:
  - تبويب "غير مدفوعة" - الأقساط غير المدفوعة بالكامل
  - تبويب "مدفوعة" - الأقساط المدفوعة بالكامل
  - تبويب "متأخرة (متابعة)" - الأقساط المتأخرة عن موعد الاستحقاق

### Changed
- تحسين شاشات المشتريات - عرض داخلي بدلاً من نوافذ منبثقة:
  - فواتير المشتريات
  - التحويلات بين المخازن
  - مرتجعات المشتريات

### Fixed
- إصلاح أسماء العلاقات في Prisma (أسماء بحرف كبير):
  - `contract` → `InstallmentContract`
  - `customer` → `Customer`
  - `invoice` → `Invoice`
  - `company` → `Company`
  - `branch` → `Branch`

|-------|-------|
| **Major (X.0.0)** | تغييرات جذرية غير متوافقة مع الإصدارات السابقة |
| **Minor (x.Y.0)** | ميزات جديدة مع الحفاظ على التوافق |

## كيفية إنشاء Release جديد

1. **تطوير التغييرات** - إتمام التطوير والاختبار
2. **تحديث CHANGELOG.md** - إضافة التغييرات تحت "[Unreleased]"
3. **إنشاء Commit** - `git commit -m "✨ v2.x.x - وصف التغيير"`
4. **إنشاء Tag** - `git tag -a v2.x.x -m "v2.x.x"`
5. **رفع لـ GitHub** - `git push origin master --tags`
6. **إنشاء Release** - من GitHub UI أو API
